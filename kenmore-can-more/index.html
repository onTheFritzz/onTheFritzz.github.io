Kenmore Can… more.

January 17, 2023

Tags

  • Appliance Repair
  • Refurbish

Q: Why are there applicances on a tech blog?

A: A welcomed consequence of taking apart laptops is that organizing the size and location of ~50 screws. Keeping track of the reverse order when dissassembling of bezels, hinges, ribbon cables and more is that many things end up not being as involved by comparison. Well… that and my family needs clean clothes.

Model: Kenmore 400 Dryer

Issues/Diagnostic

  • Machine noticeably struggles when spinning timer.
  • Machine will sometimes run endlessly. Literally.
    • A dryer that doesn’t turn off is dangerous.
    • A dryer that doesn’t turn off is expensive and not eco friendly.
  • Machine hard shuts off at specific timer intervals
    • This shows, issue is unrelated to thermistors/temp regulators in the dryer
    • Dryer shuts off when timer is exactly “50” regardless of run time.
    • Dryer does not start when timer is “started” below “40”
  • Machine will run like normal but not provide heat

Armed with this information, the sucker got popped open, cleaned up and fixed. The issue was the timer, where 240v (yes, unplug a dryer before you go poking) completes the circuit within the timer. The tiny nubs that disconnect/reconnect every cycle, likely arc and get quite hot carrying that 240v. It’s a temporary fix, to have dry, clean clothes while a replacement is shipped.

Part Number: W10185976
