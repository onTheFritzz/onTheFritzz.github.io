I AM g-ROOT

July 23, 2022

Tags

  • NAS
  • Root Access

Popped first root shell on a Buffalo TS-XLC66 Terrastation NAS. Using java and a version of ACP-Commander, a root shell monolith appeared from nothing.

It was read that Buffalo Terastations use hardware to limit root read/write access. This doesn’t appear to be the case as the root directory has been traversed and touch test commands have been issued.

SSH capabilities were also attempted but that may be where the hardware limitations come into play. For now telnet will suffice. All this to figure out why FTP transfer speeds are a measley 3MB/s.

Assumption is CPU caps, but we will see.
For now… Yo So gROOT!

Modified Bash script for Windows, that popped shell:

@ECHO OFF
set PASSWD=APassWdHere
set IPADDY=192.168.50.171
set nas=java -jar acp_commander.jar -q -t %IPADDY% -pw %PASSWD% -c
%nas% "(echo ssh.pass;echo ssh.pass)|passwd"
%nas% "sed -i 's/SUPPORT_SFTP=0/SUPPORT_SFTP=1/g' /etc/nas_feature"
%nas% "grep SFTP /etc/nas_feature"
%nas% "sed -i 's/#PermitRootLogin/PermitRootLogin /g' /etc/sshd_config"
%nas% "grep Root /etc/sshd_config"
%nas% "sed -i 's/ exit 0/ #exit 0/g' /etc/init.d/sshd.sh"
%nas% "/etc/init.d/sshd.sh stop"
%nas% "/etc/init.d/sshd.sh start"

telnet %IPADDY%

User: Root
Password: Provided Above as %IPADDY%
