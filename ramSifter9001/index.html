RAM Sorter 9001

December 1, 2022

Tags

  • Automation
  • Spreadsheets

Can’t make up my mind. RAM Sorter 9001 sounds better than RAM Sifter 9001 but it behaves just like an actual sifter.

Old way:
Manually eyeball hundreds of RAM sticks and sort them by Brand, Size and Speed. Then grab each stick a second time to sort by tiny 10+ digit alphanumerical strings. If you try to sort by referencing the trailing digits, you will be making mistakes.

New Way:
Scan the barcode that references this same alphanum string. The scanning is entered into a spreadsheet with some quick magic to literally flash different colors. Each stick gets put in the corresponding, color coded, box.

All in all, a hacked together ugly spreadsheet, that could be cleaned up however the end result is an increased accuracy from a mere 37% to 99% and takes half the time to do it. Plus a friend was made. ;D
