VFD 4 UnMe

February 2, 2023

Tags

  • NanoPi Neo

Made a fun lil weekend project outta some Noritake itron GU140x16G7000 Vacuum Flourescent Display’s that had GPIO pins. Let see what we can do.

First things first. Find the pinout of the display. Digikey please. Good thing the PDF got yoinked cuz it’s a dead link now. Here go: Datasheet

Referencing the ‘asynchronous serial’ table we know off the top which pins are positive and negative (VCC/GND). We will then assume that SIN stands for Serial IN. Which would require TX from the NanoPi Neo. Little to no history using SBUSY/NC but /RESET pin rings a bell. Lets use that for the 4th pin. Below is the correct setup for attaching the VFD to a NanoPi Neo.

VFD Function VFD Pin # Wire Color NPiNeo Pin # NPiNeo GPIO Function
VCC 1 RED 4 5V OUT
Serial IN (SIN) 2 YELLOW 8 UART1-TX
GND 3 BROWN 6 GND
RESET 6 ORANGE 10 UART1-RX

NPiNeo schematic used for reference, below.

You can also use the above table to talk to this display using a USB2UART adapter over a COM port on another computer. e.g: Windows –> Putty –> COM4 –> USB2UART Adapter –> VFD

Here are the following instructions for programatically manipulating the VFD on a NanoPi Neo running Armbian. First we install dependencies

sudo apt-get install python3-pip
sudo apt-get install python-is-python3
pip install pyserial
sudo armbian-config
  • system –> hardware –> enable UART 1,2,3
  • reboot!!!

Use python to orient the display. Here’s a quick lil test. It can work on a Windows device too but you need to change /dev/ttyS1 –> COM4 where 4 is the COM port of UART adapter on Windows machine in device manager. ie: devmgmt.msc

Default BAUD rate of 38400 discerned from manufacturer’s GUD-10K application. BAUD rate wasn’t referenced in original datasheet found and the application never did work properly but served a purpose at least.

import datetime, serial

class GU140x16G7000():
  def __init__(self):
    port = serial.Serial('/dev/ttyS1', 38400, timeout=.01)
    self.port = port

  def clearScreen(self):
    self.port.write(b'\x0C')

  def write(self, content):
    if type(content) != list:
      content = list(content)

    for x in content:
      x = bytes(x, 'utf-8')
      self.port.write(x)

  def breakTime(self):
    down = f'      BREAKTIME!    '
    clear = False
    hms = datetime.datetime.now().strftime('%H:%M:%S')
    up = f'      {hms}     '
    self.write([up, down])

vfd = GU140x16G7000()
vfd.breakTime()
